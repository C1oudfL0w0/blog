ECS入门

云服务器（Elastic Compute Service）

  1. 使用实验子账号登录云产品控制台

    子账号登录地址：https://signin.aliyun.com/login.htm#/main

  2. 在左侧列表处单击产品与服务，然后单击云服务器ECS，进入ECS管理控制台

  3. 在云服务器ECS控制台页面的左侧的导航栏中，选择实例与镜像>实例，进入实例页面

  4. 然后在实例页面顶部菜单栏中选择地域。找到实验室提供的ECS实例资源，并单击此实例ID，进入此台ECS实例的管理界面。

  5. 单击云盘，可以查看到此实例中包含的两个磁盘：20G的数据盘和80G的系统盘。

  6. 单击实例详情，进入实例操作界面，在主页面基本信息栏中，单击重置实例密码。

  7. 在重置实例密码的对话框中，输入新密码，例如Passw0rd，确认密码，然后单击保存密码。

  8. 在重置实例密码的对话框中，单击立即重启。在实例详情的页面中，可以看到实例的状态从停止中变为运行中，此时证明ECS实例已正常运行。

  9. 在实例详情页面，单击远程连接。

    在远程连接对话框的通过Workbench远程连接区域，单击立即登录。

  10. 在登录实例对话框中，输入密码，单击确定。

    于是登录成功

    提交评测完成实验任务

部署LAMP

LAMP：

  • “L”为Linux的缩写，是操作系统软件，是服务器中其它软件运行的基础，常见的CentOS、Ubuntu等；
  • “A/N”为Apache/Nginx的缩写，是Web服务器软件，用来运行网站页面以及响应处理用户请求；
  • “M”为MySQL的缩写，是数据库软件，用来存放网站运行必要的系统数据和用户数据；
  • “P”为PHP的缩写，是编程软件，如用来运行WordPress软件的代码。

Apache

# 安装 Apache
yum -y install httpd
# 设置 Apache 开机自启动
systemctl enable httpd
# 启动 Apache
systemctl start httpd
# 查看 Apache 运行状态
systemctl status httpd

Mysql

# 更新 YUM 源
rpm -Uvh http://dev.mysql.com/get/mysql57-community-release-el7-9.noarch.rpm
# 安装 MySQL 数据库
yum -y install mysql-community-server --nogpgcheck
# 设置 MySQL 开机自启动
systemctl enable mysqld
# 启动 
MySQLsystemctl start mysqld
# 查看 MySQL 运行状态
systemctl status mysqld
# 查看 MySQL root用户默认密码
grep 'temporary password' /var/log/mysqld.log |awk '{print $11}'
# 执行MySQL 数据库初始化命令
mysql_secure_installation
