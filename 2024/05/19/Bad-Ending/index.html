Bad Ending?

2024/5/19 Thinking
字数统计: 1.4k字   |   阅读时长: 4分   |   总文章阅读量：次

前言

原本写了很多很多，但是想了想没必要，于是又删掉不少

虽然大二还没结束，但是接下来的日子也是一眼望到头了，所以可以当作学年总结？

Web

在与java的搏斗中死去了…

在你面前的是：

  • java苦手，一年无突破性增长
  • 实力低于同届绝大多数web手，差距还在增大
  • 比赛必零解，复现冲ak，现在到达战场的是诸葛亮最强形态——事后诸葛亮
  • 投入 >> 产出的fw web狗！

上次阶段总结的时候提过，这个学期把java安全拿下，至少别再无从下手了

预期是这学期要学到各种java内存马的，实际上因为各种事情只学了三条cc链，能做的题只有去年国赛的 DeserBug 和 XYCTF 的那道java
而这个学期如今只剩下了一个月多一点的时间，重心得转移到预习期末考和六级上面，不怎么想留级，也不想重来
于是这学期的java安全学习基本上可以宣告再次失败了

对照一下Boogipop神的进度，依旧没能跨过他23年3月份学的东西，差距是越来越大了。。。但是没有时间能投入学习

和校队里其它方向的师傅相比，他们花的时间比我少不少，但是一直在进步变强，而我常常因为太菜而和您们格格不入x

联合战队？总不能再学一年web大三再进吧，这下算是寄了
不过，曾经有两次机会能让我接触到联合战队，但我没能把握住，连聊天与面试的机会也没能得到。。

“我从来都不觉得打web开心过”

比赛

你打ctf快两年了，当赛棍打过这么多比赛，一定很厉害吧？

  • 一刻也没有为上一个比赛的爆零哀悼，立刻赶到战场的是——复现！
  • 一刻也没有为比赛复现不完哀悼，立刻赶到战场的是——下一个比赛！

入坑以来DASCTF、NSS Round web场场报名，但是除非是特别简单的签到题，否则永远是0解，永远只有赛后复现的份；唯一的变化是赛后看了wp，反应从原来的 “看不懂不会复现” 变成了 “哦这题原来是这样入手的啊”

最近看了下西电校赛Mini-L CTF 2024，忘了打，就看了下别的师傅的复现
7道web题里面有3道java，呃，不会打shiro，没学也不会打mybatis，fastjson更是完全没去看
然后看了下排行榜，比同一届的师傅菜得多已经是既定事实了，然后23级的师傅也能够杀穿web当 java master了，唉后生可畏

今天也是刚刚结束了CISCN的初赛，评价是哈人，一部分是因为web约等于爆0了，好几题有思路但是不知道怎么做或者问题出在哪里了😭；另一部分是惊叹于黑灯1200+的提交次数😰

是的孩子们，ctf的未来靠你们了.jpg

唉不是怎么没人能教教我web啊，原来是我几乎没有认识的师傅啊

之前听一位技术员说过，网安靠的更多是圈子和人脉，现在看来确实。缺少人脉，不混圈子又能单打独斗到什么时候呢？

协会

想要为协会建设一点什么东西，可是自己的事情又做不好

缺人缺钱缺时间缺精力，很多计划最后都没下文了

很难想象巨魔卸任之后的协会会怎样。。

生活&学业

自打蓝帽杯结束之后就是长期的两点睡早八起作息，肝疼（

现在的生活已经是日趋单调了，五一假期也是在宿舍待了五天，每天基本就是起床吃早饭，游戏清个体力，看ctf题，午睡完起来看会题，然后去乐跑，晚上打会游戏，看会期中考的内容，于是就到睡觉了。。。

steam库里面堆了一堆大促买的游戏，galgame也没时间去好好玩了

重修的高数和概率论在期中考之后就因为各种原因没时间学，马上又要到期末考了，高数期中考得依托，接下来是越来越没时间了

专业课不知道考什么，也不是特别愿意背八股文。。。

结束了？

接下来学一下代审，再学点hw需要的东西，然后呢？我不知道，web的强度膨胀越来越高了，我实在是不知道要不要继续下去，再走下去也只是徒增焦虑，而且web渗透红蓝队本就非我所愿，坚持在web上只是因为“被需要着”吧？

或许真正想学的是逆向？不过这意味着一切从0开始，承认过去两年的努力打水漂，而且，缺少他人的指引，会不会又走上大量的弯路？

又或者在剩下的时间像一个普通的大学生一样去好好学习，想玩就玩想学就学想睡就睡，然后投身考研大军，但这样真的能实现自己的目标吗？

不过，我果然还是无法想象停止学习技术的自己会是什么样子的，正是因为对技术的追求才有了现在的我吧

“明知自己根本做不到放弃。 那时…我终于注意到了。我——没有办法逃走。”
