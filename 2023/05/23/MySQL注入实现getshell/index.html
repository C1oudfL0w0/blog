前言

看到[LitCTF 2023]这是什么？SQL ！注一下 ！这题的wp中出现了这种解法，顺手记下来

原理

使用 MySQL 中的outfile方法，构造 sql 语句写入webshell，通过参数执行系统命令

条件

  • 具有目录写权限
  • 知道网站根目录绝对路径
  • secure_file_priv为空或指定目录在网站目录下
  • php gpc 关闭
