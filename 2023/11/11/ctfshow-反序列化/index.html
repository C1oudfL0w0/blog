在前面加上 |, 这样的话 session 反序列化的时候 php handler 会默认把 | 前面的内容当做 key, 不会解析, | 后面的才是真正应该反序列化的 value

接下来按顺序分别在index.php和check.php传入（先访问index.php主要是为了让$_SESSION['limit'] = 1，再访问check.php是因为包含了inc.php能执行对应的类）
