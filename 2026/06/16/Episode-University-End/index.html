前言

谨以此记录我的大学四年生涯，这是一段既充实又糟糕的回忆

四年太短，来时路太长

曾经有人在群里问为什么要写博客，我毫不犹豫的回答是为了记录与回望来时的路

其实最初搭建博客也只是出于一时兴起，只是后面养成了持续更新博客的习惯后总想着需要去产出些什么东西

那么现在来回忆一下，C1oudfL0w0 的“原点”吧，不少东西其实已经在过去的 Thinking 里聊过了

Episode 0

中学时期的 0w0，是一个很喜欢玩电脑，且懂得如何解决各种电脑基础问题的家伙

不过受限于当时的认知，他每周能使用电脑的时间只有短短几个小时，在加入学校的信息社团后也没能找到一个正确的学习路线，最终没能进入到中学打 oi 的世界线中（

尽管如此，他当时便笃定自己上大学之后一定会选择计算机相关的专业，即使这家伙的数学水平可以用糟糕来形容

22 年，新高考 I 卷，那一年的数学毫无争议的难，一向排名班级倒十的他很清楚，在大家都难的情况下，自己的劣势会显著缩小，遂放平心态迎接后面的考试，结果的省排名于他而言也确实可以用超常发挥来形容

后面便是选学校和专业，具体的细节已经在一周年小结中讲过了，本着不浪费一点分数选择排名最好的计算机专业原则，最后来到了 fjnu 的网安

Episode 1-3

阴差阳错之下，被学长拐到了校网安协会 NISA 的招新群中，了解到除了 ACM 算法之外，还有着 CTF 这样一种比赛的存在，与此同时，0w0 偶然看到了他感兴趣的公司的校招岗位，CTF 则是一个明显的加分项

由此 0w0 开始学习 html、php 之类的基础知识，也是因为这一学习举动，带动了整个宿舍的舍友开始踏足网安，在面试的那天晚上举舍加入 NISA，然后又在那个国庆节接触到探姬出题的 HNCTF，点燃了对 CTF 的热情

起步时期的学习是最不顺利的，没有任何能够明确指引的学习路线，由于个人性格原因和学长之间的交流也是少之又少，所以直接狠心氪金入了 ctfshow 的 vip 开始刷题，刷到一定量有了基础感悟之后再开始转入 NSS 猛刷，现在在有了 ai 之后来看并非科学的学习路径，直接变成做题区了（

经历了磕磕绊绊的起步，0w0 也是鼓起勇气在协会群里招募队友之后，迎来了他的线下赛首战——蓝帽杯，得益于队友是取证大手子，他们一路杀进决赛，0w0 第一次坐上飞机，去到外省，去到北京，一起看天安门升旗，并且发挥他仅有的 web 知识在决赛的渗透中撕开一道入口，最后带回了一等奖

大抵也是从那开始，NISA 开始得到学院和指导老师的一定支持，在此之后的差旅报销都成为了一个固定事项

而各位队友都是很强很强的，最终我们确实做到了让 NISA 的名字渐渐出现在多个大比赛的线下赛中。虽然中间也有出现断代的情况，但是新来的后辈们仍能够在 NISA 已有的平台和人脉上，取得飞快的进步，如今已能够独当一面了，可喜可贺

学业上，虽然 0w0 经常挂科导致绩点处于毕业线上，但是托协会和竞赛的福，他的综测分依旧可观，也有幸申请到学校的三好学生与奖学金，此时于他而言校园生活已经没有更多奢求的东西了，唯一后悔的大概是各门数学经常挂科导致从一开始就打消了保研考研的念头吧（

Episode -1

0w0 的大学经历看似充实与完满，实际上恰恰相反，他并没有具体的网安学习与实践规划，也不清楚自己所学的内容对于实践有什么作用。对于工科来说，脱离实践的学习应该是大忌。

首先呢，在 CTF 上，由于个人的性格原因，0w0 两度错失加入联合战队的机会，也很少与其他师傅进行技术交流。虽然标榜自己是一位 CTF 爱好者，但是他的 web 水平直到现在也未必比一些大二的师傅强，在面对中型 CMS 的代码审计和链子挖掘上也缺少了一些挖洞的思维。这一点他非常清楚，但是长久以来都不知道如何解决。

有的师傅大一就投下 hw 简历在暑期赚米（那年 hw 生态还没被冲击），而 0w0 在大二的暑假也尝试投 hw，但是侧重 CTF 做题区的学习与简历并不能给他带来任何的优势，于是大二暑期 hw 的计划泡汤了。

而在大三下学期来临前，0w0 刚刚结束所有的挂科重修，准备选择就业的未来，此时的他仍在做梦，认为大四才是正式开始实习的阶段，在此之前还有一学期的快乐时光。事实当然不会如愿，大三下学期春招的惨状重重给了他一巴掌——被甲方公司卡简历卡笔试，被乙方公司挂面试并锐评没有技术与研究。也是从这里开始，0w0 对 CTF 水平的焦虑——java 链子没学完、新的漏洞原理没复现分析，开始转变成缺少实践研究的焦虑——没有 CVE、没有 SRC、不熟八股文与面经。

挖 CVE、SRC 都需要时间，但 0w0 此时已经没有时间能重新沉淀了，最后他只好在本地的安恒分部先干普通安服，并在三个月后感到乏味与厌倦选择离开。中间幸运的进入了字节的秋招提前批面试，但由于技术储备不足，没研究过云安全，最终止步于二面。

遗憾的是，在实习与就业这件事情上，学长与指导老师能帮上的忙都极其有限，NISA 前几届的学长在本科毕业后多选择了读研考公考编，而 0w0 的目标是去上海找个甲方厂发展。

从安恒离职后，0w0 依旧不清楚下一步该如何走，在出租屋度过最后一个月后，北京长亭抛来了橄榄枝—— CTF 安服，工作内容不同于普通安服，主要就是 CTF 比赛出题与现场支撑。被北京的待遇和工作内容所吸引，0w0 开始了他的第二份实习。

在北京的生活对 0w0 来说挺快乐的，毕竟 CTF 是他所擅长的内容，而这座大城市也有充足的二次元文娱能满足他的精神需求；但是好景不长，由于学校的毕设答辩、体测补测安排在四月份，且 ciscn&ccb 的半决赛和决赛也在三四月份需要频繁请假，所以 0w0 需要同时处理好学校毕设与公司项目这两件事。很遗憾，由于他的能力不足以处理好每件事，最终在四月底离职回到学校专心解决毕业的事情。同时，由于这一段的时间与精力安排，他完美错过了春招 qaq。

After Talk

那么，一切又回到起点，在 AI 大幅投入应用的毕业季当下，0w0 这个只会打 CTF 的赛棍臭摆子又将何去何从呢？我不知道，不过后辈们依旧相信他是一个相当有能力的 web 高手，hhh，或许他只是需要一些时间研究清楚如何把四年的 CTFer 经验转变出新的成果，然后在秋招之际蓄势待发。

不过这谈何容易，秋招也意味着要与较多的 92 本科与硕博生同台竞争，而 0w0 自认为擅长的 CTF 早就已经被 AI 实现了技术平权，即使论 AI，他现在手上能运用的也仅有 deepseek 的额度，更不用说逐年恶化的就业环境了。呃，还是祝福他能度过这个难关吧（

有时候 0w0 在想，他所仰慕的那几位师傅，为什么都能够做到比赛里出现的 Web 题手拿把掐，四年下来轻松拿捏多个大厂 offer。现在看来，应该是良好的技术氛围 + 明确的规划目标 + 足够深度的思考 + 一点点个人天赋实现的。

嗯…差不多就写到这吧，很遗憾笔者的笔力不足，没能完整清晰的描述 0w0 的四年感悟，也没能让他在毕业之际如愿以偿，只是靠着一个个虚假的希望让他继续在黑暗中摸索罢了。
